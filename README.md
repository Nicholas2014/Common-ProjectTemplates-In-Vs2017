# vs2017（v15.6.4）支持的常见项目类型集合
## 用了这么长时间的visual studio，今天突然发现有好些功能还没怎么用过，好些类型的项目也从来没有尝试过。于是决定把vs支持的项目全部建一个试试，结果就有了这60多个空项目，还不包括其余不常用的，保守估计应该不下100个，真不愧为全宇宙最强大的IDE！


## 从大的方面来说，大致可以分为以下几部分
* .NET Framework (DNF)
* .NET Core(DNC)
* 移动端 (Mobile)
* UWP (UWP)
* 标准库 (DNS)

上面几部分中对于大多数人来说最常用的也就是前面三部分，后面的基本不怎么接触到。对于.NET Framework来说，又大致可以分为控制台应用程序，桌面windows程序和web应用程序，而.NET Core目前支持的基本只有控制台应用和Web应用，而移动端方面主要有Xamarin.Android、Xamarin.iOS、Xamarin.Forms三部分。
